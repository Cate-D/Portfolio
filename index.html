Peruse Cate's Work Projects

Adoption Site

Working with a partner, we created an Animal Shelter website to practice CSS cascading styles with inherited and nested child elements. We used columns for each animal's profile and practiced with specificity and how they affect CSS selectors, learned the advantages of CSS in place of inline styling, and changed styles of elements according to cascading precedence using the Mozilla Developer Network for different selectors and properties.

Languages used:

  1. HTML
  2. CSS

Week in Review Assignment

This project encompasses a week's worth of beginning lessons on Bootstrap. Working with a partner we implemented several of the most common Bootstrap resources and styles. First we linked the Bootstrap style sheet, added a container to wrap our content nicely, added padding to dynamically adjust the page elements sizes depending on the user's viewport. We added a jumbotron and panels, and styled text with a blockquote and Bootstrap helper classes.

Languages used:

  1. CSS
  2. Bootstrap
  3. HTML
